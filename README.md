# Calculator-Simulation

This project was developed as part of an embedded systems Learning.
It simulates a basic calculator using Embedded C programming.

Features:
- Keypad based number input
- Arithmetic operations (Add, Sub, Mul, Div)
- CLCD display output

Tools Used:
- Embedded C
- STM32CubeIDE
- Proteus
